## RetroFuzz Laravel Framework

This is the latest version of the RetroFuzz web skeleton based on Laravel. Fork or clone into a new project to use. Raise issues in the issue tracker to request/discuss changes. 

Only the master branch should be used as a base for new projects, develop may be unstable.

Work on the develop branch until a new feature is complete then tag with a version release and merge into master.
